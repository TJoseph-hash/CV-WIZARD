## ğŸ“¦ Installation

### Prerequisites

- Node.js & npm
- Firebase CLI (if deploying Firebase)
- OpenAI API Key

### Clone the Repository

```bash
git clone https://github.com/TJoseph-hash/cv-wizard.git
cd cv-wizard
```

### Set Up Firebase

```bash
firebase login
firebase init
```

### Install Dependencies

```bash
npm install
```
